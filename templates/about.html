About Us


Welcome to M062 Students' Information Hub, a web-based system created to help students of Section M062 manage and access their personal and academic information easily.

This website was developed using Python as the main programming language, with HTML, CSS, SQL, GITHUB, and Visual Studio Code as supporting and backup tools. It is to make sure the website works well.

We, the BSIT students of Section M062 (set B), created this project as part of our school requirements, driven by teamwork and a desire to use technology to improve our learning experience.
